# EV-1000 Pro v0.8

Главное изменение: управление ботом переведено на inline-кнопки.

## Кнопки
- 📋 Лист
- 🟢 READY
- ⭐ ТОП
- ✅ Качество
- 💰 Банк
- 📈 Отчет
- 📊 Аналитика
- 📖 Журнал
- 🔄 Обновить

Команды сохранены как резерв, но обычному пользователю они больше не нужны.

## Railway variables
- `BOT_TOKEN`
- `OWNER_CHAT_ID`
- `READY_CHECK_INTERVAL_SECONDS=1800`
- `READY_EDGE_MIN=6`
- `READY_QUALITY_MIN=85`
- `READY_RISK_MAX=30`

После загрузки файлов в GitHub Railway автоматически запустит новый деплой.
