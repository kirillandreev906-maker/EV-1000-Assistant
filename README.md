# EV-1000 Pro v0.9

Главное изменение: управление матчами и банком переведено в кнопочную панель владельца.

## Новые возможности
- ⚙️ Панель владельца;
- ➕ пошаговое добавление матча;
- ✏️ изменение параметров матча;
- ✅ расчет завершенной ставки;
- 💰 изменение банка;
- 🗑 удаление матча;
- кнопки карточек матчей прямо в разделе «Лист».

## Railway variables
- `BOT_TOKEN`
- `OWNER_CHAT_ID`
- `READY_CHECK_INTERVAL_SECONDS=1800`
- `READY_EDGE_MIN=6`
- `READY_QUALITY_MIN=85`
- `READY_RISK_MAX=30`

Чтобы панель владельца появилась, `OWNER_CHAT_ID` должен совпадать с вашим Telegram chat ID.

После загрузки файлов в GitHub Railway автоматически выполнит новый деплой.
